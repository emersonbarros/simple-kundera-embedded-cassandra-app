simple-kundera-embedded-cassandra-app
======================================

A simple proof-of-concept of a Java application with embedded Cassandra integration, using Kundera and Guice.

## Prerequisites ##
- JDK 6
- Maven 3.0.3 or newer

## Test ##

```
mvn clean compile exec:java
```
